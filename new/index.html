Projects

Tiny Flame

Tiny Flame is a simple game developed for iOS. The objective of the game is to help the tiny flame dodge falling raindrops. Try to obtain the highest score possible!

Currently not available on the App Store.

Voice Controlled Game Companion

A school project in which I created a voice controlled companion together with three other students. In the demo the companion is able to respond to simple commands given vocally by the player. A demo video is available here. The demo was created using Unity3D and Microsoft's Cortana for the voice commands.

Virtual Reality Blocks Toy

As a school project I created a virtual version of a classic childrens toy in which you have to place blocks of different shapes in holes in which they fit. The project blog can be found here. This demo was created using Unity3D, the Leap Motion, and the Oculus Rift DK2.

Electrical Wheelchair VR Puzzel Demo

During the summer of 2016, my friends and I created a small virtual reality puzzel game for the HTC Vive. We were interested in how to approach the problem of movement in virtual reality so we decided to give using a virtual permobil a try. The puzzel game was created using Unreal Engine 4 and the HTC Vive. A video of the demo can be seen here.

God-like Third Person Concept

While creating the electrical wheelchair puzzel we also discussed creating a puzzel game from the perspective of a god looking down on a third person character. I wanted to try that concept out and created a very basic demo consisting of the new perspective and a small level using assets from the wheelchair puzzel. A shord video showing the demo can be seen here.
